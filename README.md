# Sinjapp Business Docs

Public documentation for Sinjapp Business, designed for Mintlify and backed by one manual OpenAPI source.

## Local preview

```bash
npx mintlify@latest dev
```

## Files

- `docs.json` configures Mintlify navigation and theme.
- `openapi/sinjapp-business.v1.yaml` is the source of truth for the API reference.
- `scalar.mdx` links to the deployed API reference and OpenAPI source.

## Default production URL

```text
https://docs.sinjapp.org
```
